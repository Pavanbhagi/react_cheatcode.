React.js Cheat Sheet

Quick reference guide for React (v15–v16 and hooks)

Components

import React, { Component } from 'react';
import ReactDOM from 'react-dom';

class Hello extends Component {
  render() {
    return <div>Hello {this.props.name}</div>;
  }
}

ReactDOM.render(<Hello name="John" />, document.getElementById('root'));

Props & State

  • this.props — access properties passed to the component
  • this.state — dynamic internal state
  • Update state: this.setState({ key: value })
  • With class fields (Babel): 
    state = { username: undefined };

Functional Components & Hooks

import React, { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  }, [count]);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}

Lifecycle Methods

  • Mounting: constructor → render → componentDidMount
  • Updating: shouldComponentUpdate → render → componentDidUpdate
  • Unmounting: componentWillUnmount
  • Error handling: componentDidCatch

JSX Patterns

  • Inline styles: <div style={{ margin: 0 }} />
  • Conditional rendering: {`{show && }`}
  • Fragments: <Fragment></Fragment>
  • Lists with keys: {`items.map(item =>
  • {item.name}
    • )`}
  • Dangerously set inner HTML: <div dangerouslySetInnerHTML={{__html: html}} />

PropTypes

import PropTypes from 'prop-types';

MyComponent.propTypes = {
  name: PropTypes.string.isRequired,
  age: PropTypes.number,
  callback: PropTypes.func
};
